What? How? What does it do? Definition? Example
<.(period)> Class Class selectors are denoated with a period followed by the class name A class is an attribute used to define a group of HTML elements in order to apply unique CSS rules If CSS codes <.pargraph{color:red;}>any elements with <class="pargraph"> the colour red
<#> ID ID selectors are donated with a hashtag followed by the ID name ID selectors should be used for elements used only once on the page If CSS codes <#header{color:green;}> only the < id=header> will turn green
<,(comma)> Groups of Selectors A group of selectors are coded with a <,> inbetween Groups are used for if all of the elements will be styled the same If CSS codes <h1, h2, h3 {font-weight: bold;}> all headers will become bold
